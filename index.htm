Transiciones con CSS

Permiten cambiar la propiedad de css de un estado a otro en un periodo de tiempo

Propiedades de las transiciones

  • transition-property: Define que propiedad cambiar
  • transition-duration: Define la duracion del cambio.(s o ms)
  • transition-timing-function: Controla la aceleracion o desaceleracion.
    • linear-> Velocidad constante
    • ease(por defecto)-> Comienza lenta,acelera y luego desacelera.
    • ease in -> Comienza lenta y acelera.

Transicion de color de fondo

La transicion de color de fondo en Css permite cambiar suavemente el color de dondo de un elemento cuando se activa un cambio de estado, como un ;hover. Esto crea un efecto visual agradable y fluido que mejora la experiencia del usuario al interactuar con la pagina. Por ejemplo, al pasar el cursor sobre un boton, su color de fondo puede cambiar de azul a rojo, atrayendo la atencion del usuario sin ser abrupto.
